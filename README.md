# 🖼️ Image Editor Web App

## 📌 Overview
This is a browser-based image editor built using HTML, CSS, and JavaScript. It uses the Canvas API to perform real-time image manipulation directly on the client side without any external libraries.

## 🚀 Features
- Upload and display images
- Apply basic filters (brightness, contrast, grayscale, etc.)
- Real-time image preview using Canvas
- Adjust image properties dynamically
- Download edited image

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Canvas API

## ⚙️ How It Works
- The uploaded image is rendered onto an HTML `<canvas>` element  
- JavaScript accesses the canvas context using the Canvas API  
- Pixel data is manipulated to apply filters and effects  
- The updated image is rendered instantly for preview  

## ▶️ Getting Started
1. Clone the repository  
2. Open `index.html` in your browser  
3. Upload an image  
4. Apply filters and download the edited result
